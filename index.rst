Python 从入门到深入
=========================

本文辑录了笔者在学习 Python 过程中的总结，归纳，比如针对同一问题不同场景的实现。它也很好的呈现了学习 Python 从编程到解决实际问题再到依据实际情况进行高效编程的阶梯过程。

放在这里方便笔者和他人学习和参考。文档中大部分示例使用 Python3.4 版本完成，当示例结果与系统平台相关时，通常会提供 Linux 和 Windows 两个版本的输出结果。
